# list

There are two type of list tag in the HTML including: ordered list and unordered list. 

#HTML

## ol tag

`<ol>` tag is used to list orderred list.

<ol>
	<li>First List</li>
	<li>Second List</li>
</ol>

## ul tag

`<ul>` tag is used to list unorderred list. 

<ul>
	<li>The First List</li>
	<li>The Second List</li>
</ul>
